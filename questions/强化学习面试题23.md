# 强化学习面试题23：动作价值函数和状态价值函数的关系

### 动作价值函数和状态价值函数的关系

#### 1. 基本概念

- **状态价值函数（State Value Function）**  
  表示在策略 $\pi$ 下，从状态 $s$ 开始遵循策略 $\pi$ 所获得的期望回报：  
  ```math
  V^\pi(s) = \mathbb{E}_\pi\left[ G_t \mid S_t = s \right]
  ```

- **动作价值函数（Action Value Function）**  
  表示在策略 $\pi$ 下，从状态 $s$ 采取动作 $a$，之后遵循策略 $\pi$ 所获得的期望回报：  
  ```math
  Q^\pi(s, a) = \mathbb{E}_\pi\left[ G_t \mid S_t = s, A_t = a \right]
  ```

#### 2. 两者之间的关系

状态价值函数可以表示为动作价值函数的加权平均，权重为策略 $\pi$ 在状态 $s$ 下选择各动作的概率：

```math
V^\pi(s) = \sum_{a \in \mathcal{A}} \pi(a \mid s) Q^\pi(s, a)
```

反之，动作价值函数也可以通过状态价值函数表示。根据贝尔曼方程：

```math
Q^\pi(s, a) = \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma V^\pi(s^{\prime}) \right]
```

其中：
- $P(s^{\prime} \mid s, a)$ 是在状态 $s$ 执行动作 $a$ 后转移到状态 $s^{\prime}$ 的概率，
- $R(s, a, s^{\prime})$ 是对应的即时奖励，
- $\gamma$ 是折扣因子。

将上述表达式代入 $V^\pi(s)$ 的定义，可得贝尔曼期望方程：

```math
V^\pi(s) = \sum_{a \in \mathcal{A}} \pi(a \mid s) \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma V^\pi(s^{\prime}) \right]
```

同理，动作价值函数的贝尔曼期望方程为：

```math
Q^\pi(s, a) = \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma \sum_{a^{\prime} \in \mathcal{A}} \pi(a^{\prime} \mid s^{\prime}) Q^\pi(s^{\prime}, a^{\prime}) \right]
```

#### 3. 最优情况下的关系

当考虑最优策略 $\pi^*$ 时，对应的最优状态价值函数 $V^*(s)$ 和最优动作价值函数 $Q^*(s, a)$ 满足：

```math
V^*(s) = \max_{a \in \mathcal{A}} Q^*(s, a)
```

且

```math
Q^*(s, a) = \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma V^*(s^{\prime}) \right]
```

由此可推导出贝尔曼最优方程：

```math
V^*(s) = \max_{a \in \mathcal{A}} \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma V^*(s^{\prime}) \right]
```

```math
Q^*(s, a) = \sum_{s^{\prime} \in \mathcal{S}} P(s^{\prime} \mid s, a) \left[ R(s, a, s^{\prime}) + \gamma \max_{a^{\prime} \in \mathcal{A}} Q^*(s^{\prime}, a^{\prime}) \right]
```

#### 4. 总结

- 状态价值函数是动作价值函数在策略 $\pi$ 下的期望。
- 动作价值函数可通过状态价值函数和转移概率、奖励函数表示。
- 在最优策略下，两者通过最大化操作相互关联，并满足贝尔曼最优方程。
