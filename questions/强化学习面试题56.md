# 强化学习面试题56：如何通过高斯假设下的负对数似然损失（NLL）来近似Wasserstein距离

# 如何通过高斯假设下的负对数似然损失（NLL）来近似Wasserstein距离

## 1. 背景介绍

在生成模型中，我们经常需要衡量两个概率分布之间的距离。常见的距离度量包括：

- **KL散度**（Kullback-Leibler Divergence）
- **JS散度**（Jensen-Shannon Divergence）
- **Wasserstein距离**（Earth Mover's Distance）

其中，Wasserstein距离因其良好的几何性质和对低重叠分布的有效性而受到广泛关注。然而，直接计算Wasserstein距离通常较为困难，尤其是在高维空间中。

本文探讨在**高斯分布假设下**，如何利用**负对数似然损失**（Negative Log-Likelihood, NLL）来**近似**Wasserstein距离。

---

## 2. 高斯分布下的Wasserstein距离

考虑两个高斯分布：

```math
P = \mathcal{N}(\mu_1, \Sigma_1), \quad Q = \mathcal{N}(\mu_2, \Sigma_2)
```

它们之间的**2-Wasserstein距离**具有闭式解：

```math
W_2(P, Q)^2 = \|\mu_1 - \mu_2\|^2 + \text{Tr}\left(\Sigma_1 + \Sigma_2 - 2(\Sigma_2^{1/2} \Sigma_1 \Sigma_2^{1/2})^{1/2}\right)
```

当两个分布的协方差矩阵可交换（例如对角或标量协方差），该表达式可进一步简化。

特别地，若两个分布具有**相同的协方差矩阵** $\Sigma$，则：

```math
W_2(P, Q)^2 = \|\mu_1 - \mu_2\|^2
```

这表明，在协方差固定的情况下，Wasserstein距离仅依赖于均值差异。

---

## 3. 负对数似然损失（NLL）

给定数据点 $x$ 和模型预测的分布 $Q = \mathcal{N}(\mu, \Sigma)$，其对数似然为：

```math
\log p(x; \mu, \Sigma) = -\frac{1}{2}(x - \mu)^\top \Sigma^{-1}(x - \mu) - \frac{1}{2}\log|\Sigma| - \frac{d}{2}\log(2\pi)
```

对应的**负对数似然损失**为：

```math
\text{NLL}(x; \mu, \Sigma) = -\log p(x; \mu, \Sigma)
```

在训练生成模型（如VAE）时，我们通常最小化NLL以拟合真实数据分布。

---

## 4. NLL 与 Wasserstein 距离的联系

假设真实数据分布为 $P = \mathcal{N}(\mu_p, \Sigma_p)$，模型分布为 $Q = \mathcal{N}(\mu_q, \Sigma_q)$。

若我们**固定协方差矩阵**（例如令 $\Sigma_q = \sigma^2 I$，且 $\sigma^2$ 接近 $\Sigma_p$ 的特征值），则NLL的主要项为：

```math
\text{NLL} \approx \frac{1}{2\sigma^2} \|x - \mu_q\|^2 + \text{const}
```

在期望意义下（对 $x \sim P$ 取期望）：

```math
\mathbb{E}_{x \sim P}[\text{NLL}] \approx \frac{1}{2\sigma^2} \|\mu_p - \mu_q\|^2 + \frac{1}{2\sigma^2} \text{Tr}(\Sigma_p) + \text{const}
```

注意到 $\|\mu_p - \mu_q\|^2$ 正是**协方差相同时Wasserstein距离的平方**。

因此，在以下条件下：

- 真实分布与模型分布均为高斯；
- 协方差矩阵相近或固定；
- 模型通过最小化NLL学习均值；

**最小化NLL等价于最小化Wasserstein距离的近似**。

---

## 5. 近似成立的条件与局限性

### ✅ 成立条件：

- 分布接近高斯（或局部高斯）；
- 协方差结构已知或可估计；
- 均值是主要差异来源。

### ❌ 局限性：

- 对非高斯分布（如多模态）效果差；
- 忽略协方差差异，无法捕捉分布形状变化；
- 仅近似 $W_2$ 距离，且依赖高斯假设。

---

## 6. 实际应用示例

在**变分自编码器**（VAE）中：

- 编码器输出均值 $\mu$ 和方差 $\sigma^2$；
- 解码器重构时使用高斯似然；
- 损失函数包含重构项（即NLL）和KL正则项。

若我们忽略KL项，仅优化重构损失，则：

```math
\min_\theta \mathbb{E}_{x \sim \text{data}}[\text{NLL}(x; \mu_\theta(x), \sigma^2)]
```

在理想情况下，这推动生成分布向真实分布靠近，**近似最小化Wasserstein距离**（尤其在低方差、高斯假设成立时）。

---

## 7. 总结

| 方法 | 是否依赖高斯假设 | 是否可导 | 是否近似Wasserstein |
|------|------------------|--------|---------------------|
| 直接计算 $W_2$ | 否（但闭式解需高斯） | 否（一般） | 精确 |
| NLL（高斯） | 是 | 是 | 是（在协方差相近时） |

> **结论**：在高斯分布假设下，特别是当协方差结构已知或固定时，**最小化负对数似然损失可以作为一种可微、易优化的Wasserstein距离近似**，适用于生成模型的训练。

--- 

*注：该近似不替代Wasserstein GAN等显式优化W距离的方法，但在特定场景下提供了一种简洁有效的替代方案。*
