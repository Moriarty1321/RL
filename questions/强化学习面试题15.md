# 强化学习面试题15：TD误差为什么可以作为优势函数的估计？

TD误差为什么可以作为优势函数的估计？

在强化学习中，优势函数（Advantage Function）被定义为：

```math
A^\pi(s, a) = Q^\pi(s, a) - V^\pi(s)
```

它衡量了在状态 $s$ 下采取动作 $a$ 相对于遵循策略 $\pi$ 的平均表现有多好。

TD误差（Temporal Difference Error）定义为：

```math
\delta_t = r_{t+1} + \gamma V^\pi(s_{t+1}) - V^\pi(s_t)
```

我们想说明的是：**TD误差的期望等于优势函数**，即：

```math
\mathbb{E}[\delta_t \mid s_t = s, a_t = a] = A^\pi(s, a)
```

下面进行推导：

从TD误差的定义出发：

```math
\delta_t = r_{t+1} + \gamma V^\pi(s_{t+1}) - V^\pi(s_t)
```

对其取条件期望，给定 $s_t = s$ 和 $a_t = a$：

```math
\begin{aligned}
\mathbb{E}[\delta_t \mid s_t = s, a_t = a] 
&= \mathbb{E}[r_{t+1} + \gamma V^\pi(s_{t+1}) - V^\pi(s_t) \mid s, a] \\
&= \mathbb{E}[r_{t+1} + \gamma V^\pi(s_{t+1}) \mid s, a] - V^\pi(s) \\
&= \sum_{s^{\prime}, r} p(s^{\prime}, r \mid s, a) \left[ r + \gamma V^\pi(s^{\prime}) \right] - V^\pi(s) \\
&= Q^\pi(s, a) - V^\pi(s) \\
&= A^\pi(s, a)
\end{aligned}
```

其中关键一步是：

```math
Q^\pi(s, a) = \mathbb{E}[r_{t+1} + \gamma V^\pi(s_{t+1}) \mid s_t = s, a_t = a]
```

这正是贝尔曼方程对 $Q^\pi$ 的定义。

因此，我们得出结论：

> **TD误差的期望等于优势函数**，即：
> ```math
> \mathbb{E}[\delta_t \mid s_t = s, a_t = a] = A^\pi(s, a)
> ```

这意味着，**TD误差可以作为优势函数的无偏估计**。在实际算法（如A2C、PPO等）中，我们常用TD误差 $\delta_t$ 来近似优势函数 $A^\pi(s, a)$，尤其是在单步更新的情况下。

补充说明：

- 这种估计是**无偏的**，但**方差较高**，因为 $\delta_t$ 依赖于单步采样。
- 为了降低方差，实践中常使用**n步TD误差**或**GAE（Generalized Advantage Estimation）** 来构造更优的优势函数估计。
- 尽管如此，TD误差作为优势函数的估计，其理论基础正是上述期望等式。

总结：

- TD误差：$\delta_t = r + \gamma V(s^{\prime}) - V(s)$
- 其期望：$\mathbb{E}[\delta_t \mid s, a] = Q(s, a) - V(s) = A(s, a)$
- 故：**TD误差是优势函数的无偏估计**

这使得TD误差成为策略梯度方法中估计优势函数的自然选择。
